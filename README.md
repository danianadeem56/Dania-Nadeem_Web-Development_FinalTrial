# TravelEx — Premium Umrah & International Tours
## Project Overview
### TravelEx is a modern, lightweight, and fully responsive single-page travel website built using Tailwind CSS and Vanilla JavaScript. Designed to showcase premium Umrah packages and international customized tours, the platform features a seamless Single Page Application (SPA) architecture that allows users to navigate sections smoothly without page reloads.
## Core Architecture & UI
### The platform features a seamless Single Page Application (SPA) architecture that allows users to navigate sections smoothly without page reloads. It includes an interactive custom tour planner, a structured contact form, and full mobile optimization with a dynamic hamburger menu.
## Conversion & Sales Automation
### To maximize user engagement, the website features automated WhatsApp click-to-chat integration embedded directly within package listings, alongside a persistent floating chat widget for instant customer support.
